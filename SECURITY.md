# Security Policy

## Supported Versions

Versions of LogSquirl that are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 26.03   | :white_check_mark: |
| < 26.03 | :x:                |

## Reporting a Vulnerability

To report security issues, please open a
[security advisory](https://github.com/64x-lunicorn/LogSquirl/security/advisories/new)
on GitHub (not for general support).
